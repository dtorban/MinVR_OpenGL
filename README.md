# BaseGLFW

This is an example GLFW application which downloads GLFW and runs a demo program.

### Build and Install

Build the project:
  ```
  make
  # build release
  make opt
  # build debug
  make debug
  ```

Install the project:
  ```
  make install
  ```

Run the program:
  ```
  ./build/Release/GLFW_App
  ```
